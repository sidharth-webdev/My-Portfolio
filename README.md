# 🌟 Responsive Portfolio Website

A modern, fully responsive **portfolio website** built using React, HTML, JavaScript, and TailwindCSS.  
It showcases my projects, technical skills, and achievements with a clean, minimal, and interactive design.

---

## 🚀 Features

- 💻 Fully responsive design (mobile, tablet, desktop)
- ⚛️ Built with React.js for component-based structure
- 🎨 Styled with TailwindCSS for modern UI
- 🧩 Reusable and well-structured components
- 🧭 Smooth navigation and section transitions
- 📂 Sections include:
  - About   
  - Skills  
  - Projects  
  - Contact  

---

## 🛠️ Tech Stack

- React.js
- TailwindCSS
- JavaScript (ES6+)
- HTML5
- CSS3

---

## ⚙️ Installation & Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/sidharth-webdev/My-Portfolio.git

# Navigate to the project folder
cd My-Portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
